# 📱 Portfolio App

A modern and responsive **Portfolio Mobile Application** built using Flutter to showcase personal details, skills, projects, and contact information in a clean and user-friendly interface.

---

## 🚀 Features

* 👤 About Me section
* 🛠️ Skills showcase
* 📂 Projects listing
* 📞 Contact details
* 📱 Responsive UI

---

## 🧑‍💻 Tech Stack

* Flutter
* Dart
* Material UI

---

## ⚙️ Setup & Run

```bash
git clone https://github.com/YogeshDadheechi/my_portfolio
cd YOUR_REPO_NAME
flutter pub get
flutter run
```
